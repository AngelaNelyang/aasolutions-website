# Amazing Agricultural Solution Limited Website

This is my first GitHub project.

## 📌 Description

This is an agribusiness website designed for local Tanzanian farmers.
The platform focuses on promoting sustainable farming by providing and selling organic fertilizers.

## 🚀 Features

* 🌾 Provides information about agricultural products and services
* 🛒 Allows users to explore and purchase farming products
* 🔍 Search functionality for easy product discovery
* 📱 Responsive design for both mobile and desktop users
* 👤 Simple and user-friendly interface

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

## ⚙️ Installation

```bash
git clone https://github.com/AngelaNelyang/Amazing-Agricultural-Solution-Limited.git
cd Amazing-Agricultural-Solution-Limited
```

## ▶️ Usage

Open the project folder and run `index.html` in your browser.
